# react-webpack

当jsx转js 和 es6 转es5 时babel-loader 的配置

        options: {
                presets: ['@babel/preset-react', '@babel/preset-env'],
                plugins: ['@babel/plugin-proposal-class-properties']
                }

或者在.babelrc 文件中配置：

    {
        "presets": ["@babel/preset-react", "@babel/preset-env"],
        "plugins": ["@babel/plugin-proposal-class-properties"]
    }

## 在 postcss.config.js文件中配置：

    module.exports = {  
        plugins: {  
        'autoprefixer': {browsers: 'last 5 version'}  
        }  
    }
